그 꿈은 왜 가지게 되었을까?

사용자가 직접 보고, 느낄 수 있는 Ui를 만드는 것에 대해 중학교 때부터 관심이 있어 관련 분야에 관심이 많았습니다.

직접 시스템과 이용자를 이어주는 Ui의 중요성과 대단함을 알고 그것을 만드는 것에 관심을 가지게 되었습니다.

Next

무슨 직업일까?

프론트엔드 개발자는 여러분이 지금 보고 있는 웹사이트, ‘웹’을 개발하는 웹 개발 영역 중 사용자가 눈으로 보는 영역을 구축하고, 기능을 구현하는 개발자입니다. 눈으로 보는 영역이라 함은 버튼, 입력창, 페이지 내의 애니메이션, 즉 UI(User Interface)를 의미합니다.

Next

어떤 역량이 필요할까?

웹 페이지와 같은 Ui 제작을 위해서는 필수적으로 HTML, CSS, JavaScript라는 언어를 학습해야 하고, 그 외에도 다른 소프트웨어 제작 및 보완 등을 위해 Python, C++, PHP와 같은 다양한 프로그래밍 언어도 학습하여야 합니다.

저의 포트폴리오

저는 제 꿈인 프론트엔드 개발자가 되기위해 아래와 같은 많은 작업들을 해보며 프로그래밍 경험을 쌓아 나아가고 있습니다.

저는 단순한 웹 개발자를 뛰어넘어 웹 개발 이외에도 앱과 같은 다른 Ui와 내부 소프트웨어를 제작하는 백엔드 개발자의 일도 하는 소프트웨어 개발자가 되고 싶습니다.
